## Project Overview

This website showcases the structure and design of a professional corporate site for a food distribution company.

The project focuses on:

* Clean UI layout
* Component-based architecture using React
* Responsive design for desktop and mobile devices
* Clear navigation and structured content sections

The goal was to simulate the type of website a real company might use to present its services, products, and contact information.

* Responsive navigation bar

## Getting Started

Clone the repository:

```bash
git clone https://github.com/callumLG/react-food-distribution-website.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open the local server in your browser.
